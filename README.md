# 🌱 Plateforme e-commerce durable (Projet en cours)

## 🚀 Objectif

Concevoir une plateforme e-commerce centrée sur la valorisation du bois recyclé (palettes), permettant aux utilisateurs d’acheter, personnaliser et fabriquer eux-mêmes leurs produits.

Ce projet s’inscrit dans une démarche :
- écologique (recyclage de matériaux)
- pédagogique (apprentissage DIY)
- locale (production accessible)

---

## 🧠 Concept

Le site propose :
- des produits en bois de palettes (kits, planches, produits finis)
- des options de personnalisation (dimensions, finitions, gravure)
- des plans de montage détaillés pour permettre la fabrication autonome

👉 L’objectif est de combiner **e-commerce + autonomie utilisateur**

---

## ⚙️ Fonctionnalités

- 🛍️ Catalogue produits
- 🔍 Fiche produit détaillée
- 🎛️ Personnalisation (simulation)
- 📐 Plans de montage téléchargeables
- 🧺 Panier (logique en cours / à améliorer)

---

## 🎯 Problématique

Comment permettre à un utilisateur :
